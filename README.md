## How to use
1. Install miniconda from https://docs.conda.io/en/latest/miniconda.html
2. Run Anaconda prompt
3. Run `conda env create -f /path/to/environment.yml` to create the correct python environment
4. Run `conda activate leg` to activate the environment
6. Run `python path/to/script.py`

## Data
